# 🎯 Number Guessing Game (CLI + GUI)

A fun beginner-friendly Python project that includes two versions of a Number Guessing Game:

✅ Terminal (CLI) Version  
✅ GUI Version using Tkinter  

This project is useful for learning Python basics along with simple GUI development.

---

## 🚀 Features

- 🎲 Random number generation
- 📉 Hint system (Too High / Too Low)
- 🔢 Attempt counter
- 🧠 Beginner friendly logic
- 💬 Clean and well-commented code
- 🖥 CLI + GUI versions included

---

## 🛠 Requirements

- Python 3.x
- Tkinter (already included with Python)

No extra libraries required.

---

## ▶ How to Run

### ▶ Run CLI Version

Run the terminal version using:

python cli_game.py

---

### ▶ Run GUI Version

Run the GUI version using:

python gui_game.py

---

## 🎮 Gameplay

- The computer generates a random number between 1 and 100
- Player has to guess the number
- The game gives hints:

📉 Too Low  
📈 Too High  
🎉 Correct Guess  

The game also counts the number of attempts.
