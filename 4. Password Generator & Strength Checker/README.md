# 🔐 Password Generator & Strength Checker

A simple security tool built using Python.  
This project generates strong random passwords and checks password strength.

It includes both **CLI (Terminal)** and **GUI (Tkinter)** versions.  
This project is beginner-friendly and helps in learning Python along with basic security rules.

---

## 🚀 Features

- 🔑 Random password generation
- 📏 Password length control
- 💪 Password strength checking
- 🖥 CLI version
- 🪟 GUI version using Tkinter
- 🧠 Beginner friendly code
- 💬 Fully commented program

---

## ▶ How to Run

### ▶ Run CLI Version

python cli_password.py

---

### ▶ Run GUI Version

## 💪 Strength Checking Logic

Password strength is checked using simple rules:

- ✔ Length ≥ 8 characters
- ✔ Contains uppercase letters
- ✔ Contains numbers
- ✔ Contains symbols

Stronger password = more secure password
