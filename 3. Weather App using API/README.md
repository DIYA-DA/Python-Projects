# 🌦 Weather App using API (CLI + GUI)

A real-time Weather Application built using Python and OpenWeather API.  
This project includes both **CLI (Terminal)** and **GUI (Tkinter)** versions.

The app fetches live weather data from the internet using an API and displays temperature, humidity, and weather conditions.

This is a beginner-friendly project to learn API integration in Python.

---

## 🚀 Features

- 🌍 Live weather data
- 🔗 API integration (OpenWeather)
- 🖥 CLI version
- 🪟 GUI version using Tkinter
- 🌡 Temperature display
- 💧 Humidity display
- ☁ Weather condition display
- 🧠 Beginner friendly code
- 💬 Clean and commented program

---

## 🔑 Setup API Key

To use this app, you need an API key.

Steps:

1. Create an account on openweathermap.org  
2. Generate your API key  
3. Copy the API key  
4. Paste it inside the file:

config.py

Without API key the app will not work.
