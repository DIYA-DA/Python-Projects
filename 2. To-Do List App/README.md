# 📝 To-Do List App (CLI + GUI)

A simple productivity application built using Python.  
This project includes both **Command Line (CLI)** and **GUI (Tkinter)** versions of a To-Do List manager.

The app allows you to add, delete, and store tasks using file-based storage.

This project is beginner-friendly and helps in learning core Python concepts.

---

## 🚀 Features

- ✅ Add new tasks
- ❌ Delete tasks
- 💾 File-based storage
- 🖥 CLI version
- 🪟 GUI version using Tkinter
- 🔄 Auto save tasks
- 🧠 Beginner friendly code

---

## 🛠 Requirements

- Python 3.x

Tkinter is already included with Python, so no extra installation is needed.

---

## ▶ How to Run

### ▶ Run CLI Version

Run the command:

python cli_todo.py

---

### ▶ Run GUI Version

Run the command:

python gui_todo.py

---

## 📂 Storage

All tasks are saved inside the file:

tasks.txt

This allows tasks to remain saved even after closing the program.
